# Item changes
Documentation about all item changes present in the mod.

## Reusable TMs
TM items are not consumed when used to teach a pokemon a move. The bag screen was also updated to not show the quantity of the TMs.
The change was made to leave more freedom of chose to the player when it comes to pokemon move pool. There are a lot of TMs that can only be found once in the game which can create some unavoidable sub-ideal move pools to key party pokemons.  

## Prevent TM toss
TM items can no longer be toss or given to pokemon.  
The change was made with the idea that in the future the TMs can only be found or bought once.
